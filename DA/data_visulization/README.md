# Data Visualization Practice Across Multiple Platforms

This repository shows my hands-on practice in data visualization using different platforms: **Excel**, **Google Looker Studio**, and **IBM Cognos Analytics**.

The goal of this repository is to demonstrate that I can not only create charts, but also choose suitable visualizations for different business questions, build dashboards, and turn raw data into clearer insights for decision-making.

## Platforms Covered

### 1. Excel
In the Excel section, I practiced:
- creating basic charts such as bar charts, line charts, pie charts, histograms, and funnel charts
- understanding when to use different chart types
- formatting charts clearly for better presentation
- building simple dashboards
- using pivot tables and pivot charts to summarize data

### 2. Google Looker Studio
In the Google Looker Studio section, I practiced:
- building interactive reports and dashboards
- creating scorecards and KPI-style summaries
- formatting metrics such as profit and revenue
- using pivot tables with heatmaps
- comparing business metrics across dimensions such as model, system affected, month, and sentiment
- blending and organizing data for visualization

### 3. IBM Cognos Analytics
In the IBM Cognos Analytics section, I practiced:
- building multi-panel dashboards
- creating custom calculations such as margin
- formatting business metrics
- keeping or excluding specific data points from visualizations
- applying top/bottom analysis
- creating navigation paths for drill-down analysis
- adding filters to support interactive dashboard exploration

## Business Value

Through these practices, I improved my ability to:
- translate business needs into suitable charts and dashboards
- compare performance across products, time periods, and categories
- highlight key trends, patterns, and problem areas
- design interactive visualizations that support easier analysis
- present data in a more structured and decision-friendly way

This repository reflects my growing ability to use data visualization as a tool for analysis, communication, and business understanding rather than only chart creation.
