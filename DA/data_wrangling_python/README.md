# Laptop Pricing Data Wrangling

## Scenario
This project cleans a laptop pricing dataset before later analysis. The raw data contains missing values, incorrect data types, mixed units, and categorical values that are not ready for analysis or machine learning.

## Goal
Prepare the dataset so it is easier to explore, visualize, and reuse in later notebooks.

## Dataset
https://cf-courses-data.s3.us.cloud-object-storage.appdomain.cloud/IBMDeveloperSkillsNetwork-DA0101EN-Coursera/laptop_pricing_dataset_mod1.csv
